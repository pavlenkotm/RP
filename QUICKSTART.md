# Быстрый старт

## Шаг 1: Установка Python

1. Скачайте Python 3.8+ с https://www.python.org/downloads/
2. При установке отметьте "Add Python to PATH"
3. Проверьте установку:
   ```
   python --version
   ```

## Шаг 2: Установка зависимостей

Откройте командную строку в папке программы и выполните:

```bash
pip install -r requirements.txt
```

## Шаг 3: Проверка конфигурации

Откройте `config/config.json` и убедитесь, что все пути указаны правильно:

```json
{
  "paths": {
    "excel": "D:\\Найденные_папки\\каталог_картинок_с_нагрузкой.xlsx",
    "passports": "D:\\Найденные_папки\\Паспорта",
    "template_docx": "D:\\Найденные_папки\\Пример расчетов.docx",
    "output_docs": "D:\\Найденные_папки\\Расчеты на прочность"
  }
}
```

## Шаг 4: Тестовый запуск

Сначала протестируйте на одном изделии:

```bash
python test_single.py
```

Если тест прошёл успешно, запустите полную обработку:

```bash
python main.py
```

## Проверка результатов

1. Откройте папку с результатами (указана в `config.json` → `paths.output_docs`)
2. Найдите созданный документ (формат имени: `артикул_наименование_РП.docx`)
3. Откройте документ в Microsoft Word
4. Проверьте:
   - ✓ Рамки и штампы
   - ✓ Артикул и наименование
   - ✓ Нумерация страниц
   - ✓ Изображение общего вида
   - ✓ Технические параметры
   - ✓ Нагрузки

## Частые проблемы

### "pip не распознан"
- Переустановите Python с опцией "Add to PATH"
- Или используйте: `python -m pip install -r requirements.txt`

### "Excel файл не найден"
- Проверьте путь в `config.json`
- Используйте двойные обратные слэши: `D:\\папка\\файл.xlsx`

### "Паспорт не найден"
- Убедитесь, что имя файла паспорта содержит артикул
- Проверьте путь к папке с паспортами

### Поля не обновляются
- Убедитесь, что установлен Microsoft Word
- Закройте все открытые документы Word

## Логи

Все действия программы записываются в лог-файл (путь указан в конфигурации).

Коды ошибок:
- `ERR_NO_PASSPORT` - не найден паспорт
- `ERR_NO_IMAGE` - не найдено изображение
- `ERR_PDF_PARSE` - ошибка чтения PDF
- `ERR_TEMPLATE` - ошибка работы с шаблоном

## Производительность

- ~2 секунды на изделие
- 300 изделий ≈ 10-15 минут

## Поддержка

Проверьте лог-файл для диагностики проблем.
